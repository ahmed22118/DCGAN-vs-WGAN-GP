# DCGAN-vs-WGAN-GP
Anime Face GAN — DCGAN vs WGAN-GP Generate anime faces using DCGAN and WGAN-GP models trained on the anime faces dataset.

## Dataset
[Anime Faces 64×64](https://www.kaggle.com/datasets/soumikrakshit/anime-faces)

## Platform
Trained on Kaggle with GPU T4 x2

## Demo
